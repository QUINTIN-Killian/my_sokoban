# my_sokoban

A C-based recreation of the classic Japanese puzzle game Sokoban, utilizing the ncurses library to render a dynamic terminal-based interface and handle real-time input.

This project implements the logic of the 1982 transport puzzle game, where a warehouse worker must push crates onto designated storage locations. Developed in C, it focuses on grid-based movement, collision detection, and efficient screen refreshing within a terminal environment.
